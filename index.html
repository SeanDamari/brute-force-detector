🔐 מה זה התקפת ברוט פורס (Brute Force)?

התקפת כוח גס (Brute Force Attack) היא שיטת תקיפה שבה התוקף מנסה לנחש סיסמה או אישור גישה על ידי ניסיון מאות או אלפי צירופים אפשריים עד שהוא מוצא את הצירוף הנכון.

💡 דוגמה מהחיים האמיתיים:
תאר לעצמך מישהו שמנסה לפתוח מנעול קומבינציה. הוא מנסה 0000, 0001, 0002... וכך הלאה עד שהוא מוצא את הקוד הנכון. זה בדיוק העיקרון של התקפת ברוט פורס - רק שבמחשבים זה קורה מהר הרבה יותר.

⚙️ איך הכלי עובד?

📂 שלב 1: עיבוד נתונים (Data Processing)

מה קורה: הסקריפט קורא את קובץ ה-CSV עם הלוגים של השרת, מסנן שורות לא רלוונטיות, וממיין את הנתונים לפי זמן.

למה זה חשוב: לוגים של שרת יכולים להכיל מיליוני שורות. אנחנו מתמקדים רק בשורות שקשורות להתחברות (login attempts).

def load_logs(file_path): df = pd.read_csv(file_path) return df

🔍 שלב 2: מנוע הזיהוי (Detection Engine)

מה קורה: האלגוריתם בודק כל כתובת IP בנפרד ומחפש דפוס חשוד:

  • 3+ ניסיונות כושלים (קוד 401) מאותה IP
  • בתוך חלון זמן של 5 דקות
  • ואז - התחברות מוצלחת (קוד 200/201)

זה נקרא: "Sliding Window Algorithm" - אנחנו "מחליקים" חלון זמן על פני הלוגים ובודקים כל פרק זמן.

FAILED_ATTEMPTS_THRESHOLD = 3 # כמה ניסיונות כושלים = חשוד? TIME_WINDOW_MINUTES = 5 # בכמה זמן? FAIL_CODE = 401 # קוד כישלון SUCCESS_CODES = [200, 201] # קודי הצלחה

🤖 שלב 3: אינטגרציה עם AI (AI Integration)

מה קורה: כל הנתונים שזיהינו נשלחים ל-Google Gemini (מודל AI) שמקבל את התפקיד של "אנליסט SOC".

מה ה-AI מחזיר:

  • תקציר מנהלים - מה קרה במילים פשוטות?
  • הערכת חומרה - קריטי? בינוני? נמוך?
  • המלצות - מה צריך לעשות עכשיו?
  • קישור ל-MITRE ATT&CK - למידע מקצועי נוסף

הפלט: קובץ HTML מעוצב שנראה כמו דוח אמיתי של צוות אבטחה!

💡 הערה חשובה: ה-AI לא מחליט מה חשוד - הוא רק מסביר את מה שהקוד שלנו כבר מצא. הדיוק הטכני מגיע מהאלגוריתם שכתבנו, לא מה-AI!

📋 הכנות מקדימות

🔑 קבלת מפתח API של Gemini

  1. היכנס לאתר: https://aistudio.google.com/app/apikey
  2. התחבר עם חשבון Google שלך
  3. לחץ על "Create API Key"
  4. העתק את המפתח (זה נראה כך: AIzaSy...)
  5. שמור אותו במקום בטוח!
⚠️ אבטחה: אל תשתף את המפתח שלך! זה כמו סיסמה - רק לך!

📦 התקנת Python וספריות

פתח טרמינל והרץ את הפקודות הבאות:

# בדיקה שיש Python מותקן python3 --version # התקנת הספריות הנדרשות pip install pandas --break-system-packages pip install google-generativeai --break-system-packages

הערה: הדגל --break-system-packages נדרש ב-Ubuntu לעקיפת הגנות מערכת.

📁 הכנת הקבצים

צור תיקיית עבודה וארגן את הקבצים:

# יצירת תיקייה לפרויקט mkdir brute_force_detector cd brute_force_detector # יצירת תיקיית פלט mkdir output # העתקת הקבצים: # - detector.py (הסקריפט) # - server_log.csv (הלוגים)

פתח את הקובץ detector.py בעורך טקסט ושנה את השורה:

GEMINI_API_KEY = "PASTE_YOUR_API_KEY_HERE"
